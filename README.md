# Simple Poker

## Build
----

#### Windows
> 1. Install CMake & MinGW
> 2. Setup environment variable for MinGW
> 3. Go to Poker directory
> 4. Run cmake -G "MinGW Makefiles"
> 5. Run mingw32-make.exe
> 6. Done
